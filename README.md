Magic 8 Ball (JavaScript)
A simple program that answers your yes/no questions just like the classic Magic 8 Ball toy
You ask any question, and the program randomly picks one of 6 fun answers and tells you what the "magic" says.

How does it work?
The program asks you to type in a question (example: "Will I get the job?")
It secretly picks a random number between 1 and 6
Based on that number, it gives you one of these answers:

 Yes, for sure!
 Maybe, try again later
 I don't think so
 Yes, definitely
 It's hard to say right now.
 No, not likely


What this project demonstrates?
This is a beginner JavaScript practice project that uses:

Random number generation
Getting input from a user
switch statements to choose between different outcomes
Basic console output

Fun to try with friends/family
Ask silly questions like "Will I become rich?" or "Should I eat pizza today?" and let the Magic 8 Ball decide!


Finally, it shows your question along with the answer
